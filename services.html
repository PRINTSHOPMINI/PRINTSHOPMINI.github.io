SERVICE 01

인쇄

PC에서 파일을 전송하고 복합기에서 PIN번호로 찾아 출력하는 셀프 인쇄 서비스입니다. 흑백·컬러·증명서 출력이 모두 가능하며, 결제는 카드(IC, 삼성페이, 애플페이)로 하실 수 있습니다.

흑백 A470원/면
컬러 A4250원/면
흑백 A3140원/면
컬러 A3500원/면
이용방법 6단계
1
프린터 선택
  • PC에서 파일 OPEN → 프린터 선택
  • 흑백 / 컬러 / 증명서전용(PIN9999) 선택
2
PIN번호 입력
  • 핸드폰 뒷 4자리 번호 입력
  • 증명서전용은 자동으로 9999 입력
3
복합기로 이동
  • 매장 내 복합기로 이동
  • 인쇄(PRINT) 메뉴 선택
  • 절전모드라면 터치하여 해제
4
PIN번호 검색
  • PC에서 입력한 PIN 입력 후 OK
  • 목록에서 파일 선택 → 시작 터치
용지 사이즈가 Letter면 PC에서 A4로 재설정 필요
5
과금정보 확인
  • 총 가격·페이지수·컬러 여부 확인
  • 내용 맞으면 [확인] 터치
6
카드 결제
  • 단말기 음성 나오면 IC카드 꽂기
  • 삼성페이·애플페이: 와이파이 표시 면에 터치
삼성페이·애플페이는 현대카드 마스터만 가능
💡 인쇄 꿀팁
용지 사이즈를 반드시 A4로 설정하세요 (Letter 사이즈 인식 불가)
전송 후 복합기에서 30분 내 출력하지 않으면 파일이 자동 삭제됩니다
PDF 파일로 저장 후 전송하면 폰트 깨짐 없이 출력됩니다
📋
SERVICE 02

복사

원본을 스캔부에 올리고 옵션을 설정한 뒤 카드로 결제하는 셀프 복사 서비스입니다. 흑백·컬러, A4·A3, 단면·양면, 확대·축소 등 다양한 옵션을 직접 설정하실 수 있습니다.

흑백 A470원/면
컬러 A4250원/면
흑백 A3140원/면
컬러 A3500원/면
이용방법 6단계
1
원본 올리기
  • 원본 문서를 유리판(스캔부)에 올리기
  • 원고대(ADF)에 여러 장 한번에 투입 가능
2
복사 메뉴 선택
  • 복합기 화면에서 복사(COPY) 메뉴 선택
3
옵션 설정
  • 흑백/컬러 선택
  • 용지 사이즈(A4/A3) 선택
  • 단면/양면, 부수, 배율 설정
4
금액 확인
  • 설정한 옵션 기준 금액 확인
  • 내용 맞으면 [시작] 터치
5
카드 결제
  • 단말기 음성 나오면 카드 결제
6
출력 완료
  • 출력 트레이에서 복사본 수령
  • 원본 잊지 말고 가져가세요!
📄
SERVICE 03

스캔

원본을 스캔부에 올리고 결제 후 PC 폴더에서 파일을 확인하는 셀프 스캔 서비스입니다. 스캔한 파일은 매장 내 PC에서 USB·이메일로 저장하거나 전송하실 수 있습니다.

스캔 A4100원/면
스캔 A3100원/면
이용방법 5단계
1
원본 올리기
  • 원본 문서를 유리판 또는 ADF에 올리기
2
스캔 메뉴 선택
  • 복합기 화면에서 스캔(SCAN) 메뉴 선택
  • 해상도, 파일형식(PDF/JPG) 선택
3
금액 확인
  • 스캔 매수 기준 금액 확인
4
카드 결제
  • 결제 후 스캔 시작
5
파일 확인
  • PC 스캔 폴더에서 파일 확인
  • USB로 복사 또는 이메일 전송
스캔 파일은 30분 후 자동 삭제됩니다
📠
SERVICE 04

팩스

원본을 스캔 후 팩스번호를 입력하고 결제하면 발송되며, QR코드로 발송 결과를 확인하실 수 있는 셀프 팩스 서비스입니다.

국내 팩스500원/면
국제 팩스국제팩스 안됨
📤 발신(보내기)만 가능합니다
📵 수신(받기)은 불가합니다
이용방법 6단계
1
원본 올리기
  • 보낼 문서를 ADF에 넣기
2
팩스 메뉴 선택
  • 복합기 화면에서 팩스(FAX) 메뉴 선택
3
팩스번호 입력
  • 수신처 팩스번호 입력
  • 지역번호 포함하여 입력
4
옵션 설정
  • 해상도(표준/정밀) 선택
5
금액 확인
  • 매수 기준 금액 확인 후 [시작]
6
결제 · 발송 확인
  • 카드 결제 후 발송 완료
  • QR코드 스캔으로 발송 결과 확인
✨
SERVICE 05

코팅

결제 먼저 하고 전원을 켠 뒤 예열이 완료되면 코팅지를 투입하는 셀프 코팅 서비스입니다. 사용 후 반드시 전원을 꺼주세요.

A4 코팅400원/장
A3 코팅800원/장
이용방법 4단계
1
결제 먼저!
  • 기계 옆 키오스크에서 매수 선택 후 카드 결제
⚠️ 반드시 결제를 먼저 하세요
2
전원 켜기 · 예열
  • 코팅기 전원 버튼 ON
  • 예열 완료 표시등 점등까지 대기(약 3분)
3
코팅지 투입
  • 문서를 코팅 파우치에 넣기
  • 봉합된 쪽부터 기계에 투입
4
전원 OFF
  • 사용 완료 후 반드시 전원 끄기
전원 미종료 시 다음 이용자 피해 발생

SERVICE 06

제본

전원을 켜고 예열 후 제본용지를 투입해 열제본을 완료하고 10분간 냉각하는 셀프 제본 서비스입니다. 논문, 보고서, 계약서 등에 활용하실 수 있습니다.

열제본표지1,500원/1개
이용방법 6단계
1
전원 켜기 · 예열
  • 제본기 전원 ON
  • 예열 완료 표시등까지 대기(약 5분)
2
투입구 개방
  • 뚜껑 열고 투입구 개방 상태 확인
3
용지 삽입
  • 인쇄된 문서를 제본 표지 안에 삽입
  • 페이지 순서 확인 필수!
4
기계에 투입
  • 제본 표지를 기계에 투입
  • 카드 결제 후 자동으로 작동
5
타이머 대기
  • 기계 작동 중 타이머 표시
  • 완료 신호음 대기
6
10분 냉각 후 완료
  • 꺼낸 후 평평한 곳에 놓고 10분 냉각
  • 냉각 전 힘을 가하면 분리될 수 있음
냉각 필수! 급하게 확인하면 제본 분리됩니다
