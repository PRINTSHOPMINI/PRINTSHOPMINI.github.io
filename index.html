인쇄 상세 이용방법 — 6단계
1
프린터 선택
  • PC에서 파일 OPEN → 프린터 선택
  • 인쇄 옵션 설정 후 [인쇄] 또는 [확인] 버튼 클릭
흑백출력 / 컬러출력 / 증명서전용(PIN9999) 선택
2
개인식별번호 입력
  • 핸드폰 뒷 4자리 번호 입력
  • 증명서전용(PIN9999)는 입력창 없음
  • 자동으로 [9999]가 입력되어 진행됨
3
복합기로 이동
  • 매장 내 복합기(프린터)로 이동
  • 인쇄(PRINT) 메뉴 선택
  • 화면이 꺼진 경우 터치 후 절전모드 해제
4
개인식별번호 검색 / 시작
  • PC에서 입력한 PIN번호 입력 후 OK 터치
  • 목록에서 파일 선택
  • 파란색 [시작] 버튼 터치
용지 사이즈가 Letter면 PC에서 A4로 재설정 필요
5
과금정보창 확인
  • 총 가격·페이지수·컬러 여부 확인
  • 내용이 맞으면 [확인] 터치
  • 결제 전 반드시 과금정보 확인!
6
카드 투입 결제
  • 단말기 음성 나오면 IC카드 꽂기
  • 삼성페이·애플페이는 액정 아래 와이파이 표시된 면에 터치
삼성페이·애플페이는 현대카드 마스터만 가능
📋 복사 상세 이용방법 — 6단계
1
원본 스캔부에 올리기
  • 원고이송장치에 세로로 글씨가 위로 보이게 놓기
  • 책·신분증은 유리평면에 놓기
  • 유리평면 → 왼쪽 상단 모서리에 맞춰 세로로 놓기
A3 사이즈는 짧은 면을 왼쪽 끝에 붙여서 놓기
2
복사 메뉴 선택
  • 화면이 꺼져 있으면 패널 화면 터치
  • 패널 화면에서 '복사' 메뉴 선택
3
복사 옵션 설정
  • 신분증은 '신분증복사' 탭 선택
  • 매수, 컬러/흑백, 원본 사이즈, 출력 사이즈 설정
  • 원하는 옵션 선택 후 '시작' 버튼 터치
4
금액 확인 후 결제
  • 복사 매수와 출력 사이즈 확인
  • 최종 가격 확인 후 '확인' 버튼 터치
5
카드 단말기 결제
  • 신용/체크카드 → 단말기 뒷쪽에 삽입
  • 삼성페이 → 단말기 패드에 터치
6
결제 완료 후 출력
  • 결제 후 단말기 화면에서 '취소' 터치 금지!
  • 정상 결제 후 프린팅 시작됩니다
  • 가시기 전 최종 출력 매수 꼭 확인하세요
토너 혼합 중이면 잠시 딜레이 있을 수 있음
📄 스캔 상세 이용방법 — 5단계
1
원본 스캔부에 올리기
  • 스캔할 원본을 원고이송장치에 세로로 넣기
  • 제본된 책은 복합기 윗부분 열고 유리평면 왼편에 세로로 끝 맞춰 놓기
2
스캔 메뉴 선택
  • 터치패널에서 스캔 메뉴 선택
  • 스캔 파일을 저장할 위치 선택
  • 스캔 옵션 확인 및 선택 후 시작 버튼 터치
3
금액 확인 후 결제
  • 스캔 페이지 수, 컬러/흑백 여부 확인
  • 용지 사이즈와 총 가격 확인 후 결제
스캔 가격은 컬러/흑백·사이즈 상관없이 동일
4
카드 결제
  • '카드를 대주세요' 음성 후 단말기 뒷편에 카드 꽂기
  • 삼성페이는 터치패드에 대기
  • 결제 후 단말기·복합기 패널 터치 금지!
5
PC에서 스캔파일 확인
  • PC 바탕화면에서 복합기 번호 폴더 오픈 (1번 복합기 → '1복합기스캔함' 오픈)
  • 해당 폴더의 스캔 파일은 30분 후 자동 삭제 — 반드시 메일 또는 USB로 이동!
⚠ 바탕화면 폴더를 임의로 이동하거나 삭제하지 마세요!
📠 팩스 상세 이용방법 — 6단계
1
원본 스캔부에 올리기
  • 원본을 원고이송장치에 세로로 넣기
  • 제본된 책은 복합기 윗부분 열고 유리평면 왼편에 세로로 끝 맞춰 놓기
2
팩스 메뉴 선택
  • 복합기 터치패널에서 4번째 팩스 메뉴 터치
3
팩스번호 입력
  • 지역번호 포함 팩스번호 입력
4
옵션 선택
  • 원본 사이즈, 단/양면, 원본 원고 방향 설정
  • 옵션 설정 완료 후 '시작' 버튼 터치
5
금액 및 매수 확인
  • 금액·매수 확인 후 '확인' 버튼 터치
6
결제완료 및 발송 확인
  • '카드를 대주세요' 음성 후 단말기 뒷편에 카드 꽂기
  • 삼성페이는 터치패드에 대기
  • 결제 완료 후 QR코드 스캔하여 발송 확인
※ 전송 파일이 큰 경우 도착까지 시간이 걸립니다
