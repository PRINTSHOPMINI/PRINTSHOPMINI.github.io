1Supabase 테이블 생성 (최초 1회)

Supabase 대시보드 → SQL Editor에서 아래 SQL을 실행하세요.

-- 1) 매출 자동 업로드 (북마클릿용) CREATE TABLE IF NOT EXISTS daily_sales ( id bigint generated always as identity primary key, sale_date date not null, branch text not null, total_amount integer default 0, total_count integer default 0, print_amount integer default 0, copy_amount integer default 0, scan_amount integer default 0, fax_amount integer default 0, product_amount integer default 0, created_at timestamptz default now(), UNIQUE(sale_date, branch) ); -- 2) 매입 항목 목록 CREATE TABLE IF NOT EXISTS cost_custom_items ( id bigint generated always as identity primary key, item_name text not null unique, is_default boolean default false ); -- 3) 매입 데이터 CREATE TABLE IF NOT EXISTS cost_entries ( id bigint generated always as identity primary key, year_month text not null, branch text not null, item_name text not null, amount integer default 0, entry_date date, updated_at timestamptz default now(), UNIQUE(year_month, branch, item_name) ); -- 4) 매출 수동 입력 항목 목록 CREATE TABLE IF NOT EXISTS sales_custom_items ( id bigint generated always as identity primary key, item_name text not null unique ); -- 5) 매출 수동 입력 데이터 CREATE TABLE IF NOT EXISTS sales_manual_entries ( id bigint generated always as identity primary key, year_month text not null, branch text not null, item_name text not null, amount integer default 0, entry_date date, updated_at timestamptz default now(), UNIQUE(year_month, branch, item_name) ); -- RLS 설정 (전체 테이블) DO $$ DECLARE t text; BEGIN FOR t IN SELECT unnest(ARRAY['daily_sales','cost_custom_items','cost_entries','sales_custom_items','sales_manual_entries']) LOOP EXECUTE 'ALTER TABLE '||t||' ENABLE ROW LEVEL SECURITY'; EXECUTE 'CREATE POLICY IF NOT EXISTS anon_all ON '||t||' FOR ALL TO anon USING (true) WITH CHECK (true)'; END LOOP; END $$; -- 기본 매입 항목 삽입 INSERT INTO cost_custom_items (item_name, is_default) VALUES ('임차료',true),('관리비',true),('통신비',true),('코팅지',true), ('용지',true),('제본표지',true),('쫄대파일',true),('L홀더',true), ('서류봉투',true),('제본기',true),('코팅기',true),('사무용품',true) ON CONFLICT (item_name) DO NOTHING;
👉 방법: supabase.com → 프로젝트 선택 → 왼쪽 SQL Editor → New Query → 위 코드 붙여넣기 → Run
2북마클릿 설치 (최초 1회)

아래 버튼을 마우스로 드래그해서 브라우저 북마크 바에 끌어다 놓으세요.

🔄 프린트샵미니 매출 업로드
  • 북마크 바가 안 보이면: 브라우저에서 Ctrl+Shift+B (표시/숨기기)
  • 기존 북마클릿이 설치되어 있다면 반드시 삭제 후 재설치하세요 (선불제외금액 반영)
  • 드래그가 안 되면: 위 버튼 우클릭 → "즐겨찾기에 추가" 후 주소 란에 아래 코드를 붙여넣으세요
▶ 북마클릿 코드 직접 보기 (드래그 안될 때)
3매일 사용 방법
  • ① fax.printnuri.com/payment 접속 (이미 로그인된 상태)
  • ② 날짜 범위 설정 후 그룹명 프린트샵미니 선택하고 검색
  • ③ 북마크 바에서 🔄 프린트샵미니 매출 업로드 클릭
  • ④ "✅ 업로드 완료!" 알림이 뜨면 이 페이지에서 새로고침으로 확인
