1Supabase 테이블 생성 (최초 1회)

Supabase 대시보드 → SQL Editor에서 아래 SQL을 실행하세요.

CREATE TABLE daily_sales ( id bigint generated always as identity primary key, sale_date date not null, branch text not null, total_amount integer default 0, total_count integer default 0, print_amount integer default 0, copy_amount integer default 0, scan_amount integer default 0, fax_amount integer default 0, product_amount integer default 0, created_at timestamptz default now(), UNIQUE(sale_date, branch) ); ALTER TABLE daily_sales ENABLE ROW LEVEL SECURITY; CREATE POLICY "anon_all" ON daily_sales FOR ALL TO anon USING (true) WITH CHECK (true);
👉 방법: supabase.com → 프로젝트 선택 → 왼쪽 SQL Editor → New Query → 위 코드 붙여넣기 → Run
2북마클릿 설치 (최초 1회)

아래 버튼을 마우스로 드래그해서 브라우저 북마크 바에 끌어다 놓으세요.

🔄 프린트샵미니 매출 업로드
  • 북마크 바가 안 보이면: 브라우저에서 Ctrl+Shift+B (표시/숨기기)
  • 드래그가 안 되면: 위 버튼 우클릭 → "즐겨찾기에 추가" 후 주소 란에 아래 코드를 붙여넣으세요
▶ 북마클릿 코드 직접 보기 (드래그 안될 때)
3매일 사용 방법
  • ① fax.printnuri.com/payment 접속 (이미 로그인된 상태)
  • ② 날짜를 오늘로 설정하고 그룹명 프린트샵미니 선택 후 검색
  • ③ 북마크 바에서 🔄 프린트샵미니 매출 업로드 클릭
  • ④ "✅ 업로드 완료!" 알림이 뜨면 이 페이지에서 새로고침으로 확인
⏰ 매일 업무 끝나고 한 번만 클릭하면 그날 데이터가 자동 저장됩니다.
같은 날짜 데이터를 여러 번 올려도 중복 저장 없이 자동 업데이트됩니다.
