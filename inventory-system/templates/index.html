미인증 상태

아래 방법 중 하나로 인증하면 매출 데이터를 연동할 수 있습니다.

방법 1 아이디 / 비밀번호 로그인

백엔드가 printnuri.com에 직접 로그인하여 세션 쿠키를 획득합니다.

방법 2 세션 쿠키 직접 입력

브라우저에서 printnuri.com 에 로그인한 뒤,
개발자도구(F12) → Network 탭 → 아무 요청 클릭 → Request Headers → Cookie 값을 복사하여 붙여넣으세요.

예: PHPSESSID=abc123def456; ci_session=xyz789uvw012

🔖 재고 업로드 북마클릿 설치 최초 1회만 설치

아래 버튼을 마우스로 드래그해서 브라우저 즐겨찾기 바에 끌어다 놓으세요.
설치 후 fax.printnuri.com/tossOrderItem 페이지에서 클릭하면 자동 업로드됩니다.

📦 재고 업로드 (드래그해서 설치)

• 즐겨찾기 바가 안 보이면 Ctrl+Shift+B

• 사용법: fax.printnuri.com → tossOrderItem → 날짜 설정 후 검색 → 북마클릿 클릭

• 월별로 나눠서 실행 권장 (데이터 누락 방지)

▶ 코드 직접 복사 (드래그 안 될 때)

⚠️ 스크래퍼 조정 안내

• scraper.py 의 LOGIN_URL과 SALES_BASE_URL을 실제 사이트 경로로 교체하세요.

• 로그인 폼의 실제 input name 속성값에 맞게 login_data 딕셔너리 키를 수정하세요.

• 판매 내역 테이블의 HTML 구조에 맞게 parse_sales_data() 의 컬럼 인덱스를 확인하세요.
